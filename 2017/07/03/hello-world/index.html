Hello World

17-07-03
  • 杂

你好，世界！

始

按照昨天计划，今天搭建 blog 。blog 托管毫无悬念的选择 Hexo + github pages 。然后从以前 firefox 收藏夹中翻出一个珍藏好久的主题 ：hexo-theme-indigo 。实话这主题做的太棒了，处处合我心意。

由于昨天学了一点 Git ，今天在 github 建仓库，部署等异常顺利。然后就去看 hexo 文档。由于作者是台湾人，文档罕见的提供简体中文版，我就英文中文对照着来看了（英文差真是要命）。文档说的比较笼统，自己水平又低，就打开配置文件，改一点本地看一下效果。后来后面一大片英文配置项，文档里又没提到，就索性不管了。

评论

blog 顺利跑起来了。头疼的是评论。多说显然是最好的选择，可是不久前倒闭了。恕我直言其他评论全是辣鸡。以前我瞎逛看见过 gitment ，这次在这个主题里又发现提供了 gitment 的配置项！真是神级主题！就这样乐呵呵的去学着配置 gitment 。

gitment 有一些不好：

  • 评论需要 github 账号，这对一些人来说是个门槛。
  • 每条评论相当于在 github 提交一个 issue ，一来占用 github 资源，二来有些开了邮件提醒的用户会比较烦。

对于占用资源这件事，gitment 作者邮件咨询过 github 。github 回信展现的精神真是让人佩服：

We’re pleased to see you making use of the tools and resources available on GitHub.

跟着教程糊里糊涂的把 gitment 跑了起来。中间遇到过麻烦，卡了好久。

终

配置了几个零零碎碎的图片，这主题这是强大！
接着是熟悉一些 Markdown 语法，熟悉一下 hexo 的标签、分类、发文。
以后还需要学习一下 七牛 的图床。
先这样吧。
